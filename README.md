

# flutter_Projcets

本项目汇集多个Flutter项目，适用于Flutter新手练习使用。
项目中使用的接口为玩Android接口及国外开源接口。感谢开源人员对开发社区做出的贡献！

项目中使用的插件：

 ```
  provider: ^5.0.0
  flustars: ^2.0.1
  flutter_screenutil: ^5.0.0
  http: ^0.13.3
  json_serializable: any
  image_fade: ^0.2.1
  get: ^4.1.4
  dio: ^4.0.0
  logger: ^1.0.0
  webview_flutter: ^2.0.7
  shared_preferences: ^2.0.5
  sqflite: ^2.0.0+3
  path: ^1.8.0
```
